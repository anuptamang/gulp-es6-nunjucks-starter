# Requirements
* Node >= 8.11.13
# Installation
```sh
$ npm i -g gulp-cli
```
```sh
$ npm i
```
# Start
```sh
$ gulp
```
# Build
```sh
$ gulp build
```
# Build for QA
```sh
$ gulp qa
```
